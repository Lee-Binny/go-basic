# go-chat
웹 소켓을 사용한 실시간 채팅 애플리케이션입니다.  
사용 언어: React, Golang(gin, gorilla)  
운영 체제: Mac OS  
사용 툴: visual sudio code  

### Install
```shell
go get -u "github.com/gin-gonic/gin"
go get -u "github.com/gorilla/websocket"
```

### Run Project
```shell
cd client
npm i
npm run build

cd ../server
go run .
```
