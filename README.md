# go-chat
The project to create a real-time chat application using Golang(gin, gorilla) and React.

### Install
```shell
go get -u "github.com/gin-gonic/gin"
go get -u "github.com/gorilla/websocket"
```

### Run Project
```shell
cd client
npm i
npm run build

cd ../server
go run .
```
http://localhost:3030/
